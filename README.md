# dele - 三角洲痕迹清理工具

三角洲游戏痕迹清理工具，集成高级自毁机制和 Root 环境隐藏功能。

## 主要功能

### 1. 下发文件检测
检测风险文件和监控痕迹

### 2. 深度环境监测
检测设备安全状态和潜在风险

### 3. 清理文件部分
执行基础文件和数据清理

### 4. 设备硬件标识变更
修改设备指纹和网络标识

### 5. 全维深度核心清理
一键执行清理和标识变更

### 6. Root 环境隐藏方案 ✨新功能
- 🔍 Root 环境检测（支持 Magisk/KernelSU/APatch）
- 📋 查看推荐的隐藏模块搭配方案
- ⚡ 一键配置隐藏环境
- 🎮 三角洲行动专项隐藏配置
- 🧹 Root 痕迹深度清理
- 📦 隐藏模块下载地址汇总

详细使用说明请查看：[ROOT_HIDING_GUIDE.md](ROOT_HIDING_GUIDE.md)

## 使用要求

- Android 设备
- Root 权限
- 支持的 Root 方案：Magisk 26+、KernelSU、APatch

## 使用方法

```bash
# 获取 Root 权限
su

# 运行脚本
sh dele.sh
```
