# dele
腾讯游戏痕迹清理工具 - 多游戏智能清理版

## 功能特性

### 主要功能

1. **下发文件检测** - 检测风险文件和监控痕迹
2. **深度环境监测** - 检测设备安全状态和潜在风险
3. **清理文件部分** - 执行基础文件和数据清理
4. **设备硬件标识变更** - 修改设备指纹和网络标识
5. **全维深度核心清理** - 一键执行清理和标识变更
6. **多游戏智能清理** ⭐ (新增) - 智能识别并清理多款腾讯游戏数据

## 智能清理特性

### 关键词识别系统

工具采用智能目录扫描技术，自动识别并清理以下类型的目录：

**高风险目录（风控相关）**
- `turing` - 图灵设备指纹/反欺诈
- `qimei` - 齐眉设备ID
- `beacon` - 灯塔SDK行为分析
- `ano_tmp` - ACE反作弊下发文件
- `apm` - 性能监控
- `hawk` - Hawk监控
- `crash` - 崩溃记录
- `jwt_token` - JWT令牌
- `itop_login` - 登录信息

**缓存目录**
- `cache` - 缓存
- `tbs` - TBS浏览内核
- `webview` - WebView
- `tmp/temp` - 临时文件
- `log` - 日志文件
- `dex/odex` - DEX缓存
- `textures` - 纹理缓存

**SDK目录**
- `midas` - 支付SDK
- `msdk` - 腾讯MSDK
- `gcloud` - GCloud SDK
- `gvoice` - 语音SDK

## 使用说明

### 系统要求
- Android系统
- Root权限
- 支持的终端环境（Termux、MT管理器等）

### 运行方式
```bash
# 获取root权限
su

# 运行脚本
bash dele.sh
```

### 多游戏清理使用

1. 在主菜单选择 `[6] 多游戏智能清理`
2. 系统自动检测已安装的支持游戏
3. 选择要清理的游戏：
   - 输入序号清理单个游戏
   - 输入 `A` 清理全部已安装游戏
   - 输入 `0` 返回主菜单
4. 确认操作后开始清理
5. 查看清理结果和统计信息

## 技术特点

- ✅ 智能目录识别 - 不依赖硬编码路径
- ✅ 关键词匹配清理 - 自动识别风控相关目录
- ✅ 自动游戏检测 - 只显示已安装的游戏
- ✅ 批量处理支持 - 可一次性清理多个游戏
- ✅ 安全确认机制 - 所有危险操作需确认
- ✅ 实时进度反馈 - 清楚显示清理过程
- ✅ 错误处理完善 - 清理失败时给出提示
- ✅ 性能优化 - 缓存包列表减少系统调用

## 安全警告

⚠️ **重要提示**
- 本工具会删除游戏数据，请谨慎使用
- 清理后可能导致游戏需要重新登录
- 建议在使用前备份重要数据
- 修改设备标识可能影响其他应用
- 使用本工具风险自负
